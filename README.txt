This server uses the built-in http module, that's all
 - No Express
 - Nothing installed with npm


To see a list of node's built-in modules:  
$ node
Then hit <tab>, sometimes twice.  You will see http, fs, url, and
a bunch of other modules. 

 
To run this server:  
$ node index.js

Then do a get request from browser or postman:
localhost:8000
